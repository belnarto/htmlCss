By Belnarto. Published January 3rd.

Please contact Troy McClure for questions about this article.

The article element represents an independent article in a web page. It should only wrap content that can be plucked out of your page and distributed in a completely different context.

They have their own headers, footers, and document outline that are completely isolated from the rest of your site.

This fake article was written by somebody at InternetingIsHard.com, which is a pretty decent place to learn how to become a web developer. This footer is only for the containing <article> element.

Section description

The section element is sort of like an article, except it doesn’t need to make sense outside the context of the document.

That is, an app like Flipboard wouldn’t try to pull out all the section’s of your page and present them as independent pieces of content.
